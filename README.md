# ZCUS Alberta Website

A static nonprofit website for Zimbabwe Cultural Society of Alberta.

## Pages

- `index.html`
- `about.html`
- `programs.html`
- `aify.html`
- `events.html`
- `membership.html`
- `volunteer.html`
- `contact.html`
- `404.html`

## Local Preview

From this directory:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

No build step is required.
